### Start the local web server

Execute the npm script **clear** followed by **start** in the website sub-project,
or execute the following within the project folder:

```sh
npm --prefix ~/Work/xpack/xpm-js.git/website run clear
npm --prefix ~/Work/xpack/xpm-js.git/website run start
```

Navigate to the **Maintainer Info** page,
the **[Start the local web server](http://localhost:3000/xpm/docs/maintainer/#start-the-local-web-server)** section.
