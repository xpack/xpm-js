### Prepare the website

The documentation site is built using [Docusaurus](https://docusaurus.io) and
published on the project
[GitHub Pages](https://xpack.github.io/xpm/).

- Switch to the `development` branch
- Execute the **website-generate-commons** npm script in the `website/package.json`
  ```sh
  npm --prefix ~/Work/xpack/xpm-js.git/website run website-generate-commons
  ```
- **Commit** all changes
