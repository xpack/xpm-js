### Prepare the release


- Switch to the `development` branch
- `npm run fix`
- If there are changes, **commit** all changes to the `development` branch
- `npm run test`
- Check the latest commits `npm run git-log`
- Update `CHANGELOG.md`, add a line such as _**\* v0.21.3 published on npmjs.com**_
- **Commit** with the message _**CHANGELOG: publish npm v0.21.3**_
- Check the list of files included in the npm package:
  ```sh
  npm run npm-pack
  ```
- Possibly update `.npmignore`
